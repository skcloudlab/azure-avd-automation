# avd-sessionhost

Deploys a sample Azure Virtual Desktop session host VM and registers it with the host pool.

## Inputs

- vm_name
- location
- resource_group_name
- subnet_id
- vm_size
- host_pool_name
- registration_token

## Outputs

- vm_id
- vm_name
- nic_id

## Notes

This module is a sanitized sample for learning and architecture reference.
