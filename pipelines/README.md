# Pipeline Samples

This folder contains sanitized Azure DevOps pipeline examples for the Azure Virtual Desktop automation reference implementation.

## Included Pipelines

- `terraform-apply.yml`
- `terraform-destroy.yml`

## Purpose

These pipelines demonstrate how infrastructure deployment and cleanup can be executed through Azure DevOps using Terraform.

## Notes

These examples are simplified for public documentation and learning.

They do not include:

- production service connection names
- internal project identifiers
- enterprise secrets
- organization-specific deployment logic
