# Logic App Samples

This folder contains simplified Azure Logic App workflow definitions for the Azure Virtual Desktop automation reference implementation.

## Included Workflows

- `onboarding-workflow.json`
- `offboarding-workflow.json`

## Purpose

These workflow samples demonstrate how Azure Logic Apps can act as the orchestration layer between:

- ServiceNow requests
- Azure DevOps pipelines
- Terraform-based infrastructure deployment

## Notes

These files are intentionally simplified and sanitized for public documentation and learning.

They do not include:

- enterprise API endpoints
- production authentication details
- customer-specific business logic
- internal callback implementations
